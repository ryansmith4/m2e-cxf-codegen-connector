# m2e-cxf-codegen-connector

## Overview
This project provides an implementation of an m2e lifecycle extension plug-in for m2e 1.x users of Apache's cxf-codegen maven plugin. 

The project was originally hosted at [Eclipse Labs](https://code.google.com/a/eclipselabs.org/p/m2e-cxf-codegen-connector/) and migrated to GitHub when Eclipse Labs was shutdown.
The official repository for this project is https://github.com/ryansmith4/m2e-cxf-codegen-connector
